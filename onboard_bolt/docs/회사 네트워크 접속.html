개요

업무 생산성 향상(Productivity Solutions Inc.)은 직원들이 안전하고 효율적으로 회사 네트워크에 접속할 수 있도록 명확한 지침을 제공합니다. 이 문서는 회사 네트워크 접속 방법, 보안 규칙, 비상 시 조치 등에 대해 설명합니다.

네트워크 접속 방법

사내 네트워크 접속

사내에서 회사 네트워크에 접속하려면 다음 지침을 따릅니다:

• Wi-Fi 접속: 사내 Wi-Fi 네트워크에 접속하기 위해 제공된 SSID와 비밀번호를 사용합니다.

• SSID: Productivity_Network

• 비밀번호: 회사 정책에 따라 정기적으로 변경 (변경 시 인사팀 공지)

• 유선 접속: 이더넷 케이블을 사용하여 네트워크에 접속합니다.

• 연결 방법: 이더넷 케이블을 컴퓨터의 네트워크 포트와 사무실의 네트워크 소켓에 연결합니다.

원격 네트워크 접속

원격 근무 시 안전하게 회사 네트워크에 접속하기 위해 VPN을 사용합니다:

• VPN 클라이언트 설치: 회사에서 제공하는 VPN 클라이언트를 설치합니다.

• 설치 파일: IT 지원팀에서 제공

• VPN 접속 정보: VPN 서버 주소와 인증 정보를 사용하여 접속합니다.

• VPN 서버 주소: vpn.productivitysolutions.com

• 인증 정보: 사내 계정 정보 사용

네트워크 접속 시 유의사항

• 암호화된 접속 사용: 가능하면 HTTPS, SSH 등 암호화된 접속 방법을 사용합니다.

• 정기적인 비밀번호 변경: 네트워크 접속 비밀번호는 정기적으로 변경해야 합니다.

• 외부 네트워크 사용 제한: 공용 Wi-Fi 및 보안이 취약한 네트워크 사용을 자제합니다.

보안 규칙

계정 및 인증 관리

• 개인 계정 사용: 각 직원은 개인 계정을 사용하여 네트워크에 접속해야 합니다. 계정을 공유하거나 타인의 계정을 사용하는 것은 금지됩니다.

• 강력한 비밀번호 사용: 비밀번호는 최소 8자 이상, 대문자, 소문자, 숫자, 특수 문자를 포함해야 합니다.

• 2단계 인증: 회사 네트워크 접속 시 2단계 인증을 필수적으로 사용합니다.

데이터 보호

• 기밀 정보 보호: 회사 네트워크를 통해 기밀 정보를 주고받을 때는 반드시 암호화된 방법을 사용합니다.

• 외부 저장 장치 사용 제한: 외부 USB 드라이브, 외장 하드디스크 등의 사용을 제한하며, 사용 시 IT 지원팀의 사전 승인을 받습니다.

소프트웨어 사용

• 승인된 소프트웨어 사용: 회사 네트워크에서 사용하는 모든 소프트웨어는 IT 부서의 승인을 받아야 합니다.

• 정기 업데이트: 사용 중인 소프트웨어 및 운영체제는 최신 보안 패치를 유지합니다.
