비밀번호 생성

비밀번호 생성 규칙

안전한 비밀번호를 생성하기 위해 다음 규칙을 준수해야 합니다:

• 최소 길이: 비밀번호는 최소 12자 이상이어야 합니다.

• 문자 조합: 대문자, 소문자, 숫자, 특수문자(!, @, #, $, %, ^, &, *)를 포함해야 합니다.

• 예시: SecurePa$$w0rd!23

금지된 비밀번호

다음과 같은 비밀번호는 사용이 금지됩니다:

• 일반적인 단어: password, 123456, qwerty 등 쉽게 추측할 수 있는 단어

• 개인 정보: 이름, 생년월일, 전화번호 등 개인 정보 포함

• 반복 문자: aaa111, 123123 등 반복되는 패턴

비밀번호 변경

변경 주기

비밀번호는 정기적으로 변경해야 합니다:

• 변경 주기: 최소 3개월마다 비밀번호를 변경해야 합니다.

• 알림: 비밀번호 만료 1주일 전에 이메일 알림을 발송합니다.

변경 절차

  1. 로그인: 회사의 IT 시스템에 로그인합니다.

  1. 비밀번호 변경: ‘비밀번호 변경’ 메뉴를 선택합니다.

  1. 새 비밀번호 입력: 새로운 비밀번호를 입력하고 확인합니다.

  1. 변경 완료: 변경 완료 후, 이메일로 확인 메일을 받습니다.

비밀번호 보안 규칙

비밀번호 보안 유지

비밀번호의 보안을 유지하기 위해 다음 규칙을 준수해야 합니다:

• 비밀번호 공유 금지: 비밀번호를 타인과 공유하지 않습니다.

• 기록 금지: 비밀번호를 메모지, 문서 등에 기록하지 않습니다.

• 공공장소 사용 제한: 공공장소나 공용 컴퓨터에서 로그인할 때는 주의합니다.

의심스러운 활동 감지

• 이상 징후: 비밀번호 도용이 의심되는 활동을 감지한 경우 즉시 IT 보안팀에 보고합니다.

• 비밀번호 변경: 의심스러운 활동 감지 시 즉시 비밀번호를 변경합니다.

비밀번호 관리 도구

비밀번호 관리 프로그램 사용

안전한 비밀번호 관리를 위해 비밀번호 관리 도구를 사용합니다:

• 권장 도구: LastPass, 1Password, Bitwarden 등

• 사용 방법: 비밀번호 관리 도구에 비밀번호를 저장하고, 필요할 때 자동 입력 기능을 사용합니다.

• 마스터 비밀번호: 비밀번호 관리 도구의 마스터 비밀번호는 강력하게 설정하고, 정기적으로 변경합니다.
