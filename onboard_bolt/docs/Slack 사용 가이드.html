개요

슬랙(Slack)은 팀 간의 실시간 메시징과 협업을 위한 도구입니다. 이 가이드는 슬랙을 처음 사용하는 직원들을 위해 주요 기능과 사용 방법을 설명합니다.

슬랙 시작하기

계정 생성 및 로그인

  1. 초대 이메일 확인: 회사에서 발송한 슬랙 초대 이메일을 확인합니다.

  1. 계정 생성: 초대 이메일의 링크를 클릭하여 계정을 생성합니다.

  1. 로그인: 생성한 계정으로 슬랙에 로그인합니다.

  1. 워크스페이스 선택: “Productivity Solutions Inc.” 워크스페이스를 선택합니다.

프로필 설정

  1. 프로필 사진 추가: 오른쪽 상단의 프로필 아이콘을 클릭하고, “Profile & account”를 선택합니다. “Edit Profile”을 클릭하여 프로필 사진을 업로드합니다.

  1. 프로필 정보 입력: 이름, 직책, 연락처 등 프로필 정보를 입력합니다.

주요 기능 사용 방법

채널 사용

채널은 특정 주제나 프로젝트에 대한 그룹 채팅방입니다.

  1. 채널 생성

• 사이드바의 “Channels” 섹션에서 “+” 아이콘을 클릭합니다.

• 채널 이름과 설명을 입력하고, 공개(Everyone) 또는 비공개(Invite-only)를 선택합니다.

• “Create Channel”을 클릭하여 채널을 생성합니다.

  1. 채널 참여

• 사이드바의 “Channels” 섹션에서 “Browse all channels”을 클릭합니다.

• 참여하고 싶은 채널을 검색하고, 채널 이름을 클릭한 후 “Join Channel”을 클릭합니다.

메시지 보내기

  1. 채널 메시지

• 채널 목록에서 원하는 채널을 선택합니다.

• 하단의 메시지 입력창에 텍스트를 입력하고, Enter 키를 눌러 메시지를 보냅니다.

  1. 다이렉트 메시지

• 사이드바의 “Direct messages” 섹션에서 “+” 아이콘을 클릭합니다.

• 대화를 시작할 동료의 이름을 입력하고, “Start a conversation”을 클릭합니다.

• 메시지 입력창에 텍스트를 입력하고, Enter 키를 눌러 메시지를 보냅니다.

파일 공유

  1. 파일 업로드

• 메시지 입력창 아래의 클립 아이콘을 클릭합니다.

• 업로드할 파일을 선택하거나 드래그 앤 드롭하여 파일을 업로드합니다.

• 파일을 업로드하고 나면 메시지와 함께 파일이 공유됩니다.

  1. 파일 검색

• 사이드바의 “Files” 섹션을 클릭하여 공유된 파일들을 확인할 수 있습니다.

멘션 및 알림

  1. 멘션

• 특정 사람을 멘션하려면 메시지 입력창에 “@“를 입력한 후, 멘션할 사람의 이름을 입력하고 선택합니다.

• 예: @김철수, 이 프로젝트에 대해 논의가 필요합니다.

  1. 알림 설정

• 오른쪽 상단의 프로필 아이콘을 클릭하고, “Preferences”를 선택합니다.

• “Notifications” 섹션에서 알림 설정을 조정합니다. 예: 모든 새로운 메시지, 멘션 및 키워드, 다이렉트 메시지 등.

이모지 반응

  1. 이모지 추가

• 메시지 위에 마우스를 올리고, 메시지 오른쪽에 나타나는 웃는 얼굴 아이콘을 클릭합니다.

• 원하는 이모지를 선택하여 메시지에 반응을 추가합니다.

검색 기능

  1. 메시지 검색

• 슬랙 상단의 검색창에 검색어를 입력하고 Enter 키를 누릅니다.

• 검색 결과에서 원하는 메시지를 클릭하여 확인합니다.

  1. 필터 사용

• 검색 결과 페이지에서 날짜, 사람, 채널 등을 기준으로 검색 결과를 필터링할 수 있습니다.

통합 기능

  1. 앱 추가

• 사이드바의 “Apps” 섹션에서 “+” 아이콘을 클릭합니다.

• 추가하고 싶은 앱을 검색하여 설치합니다. 예: Google Drive, Trello, Zoom 등.

  1. 앱 사용

• 통합된 앱의 기능을 슬랙 채널 내에서 바로 사용할 수 있습니다. 예: Google Drive에서 파일 공유, Zoom 회의 생성 등.
