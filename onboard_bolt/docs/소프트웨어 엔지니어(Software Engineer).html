직무 설명

소프트웨어 엔지니어는 당사의 주요 제품과 서비스 개발을 담당하며, 고품질의 코드를 작성하고 유지하는 역할을 수행합니다. 이들은 최신 기술과 방법론을 사용하여 효율적이고 확장 가능한 소프트웨어 솔루션을 설계합니다.

기대되는 역할

• 코드 작성 및 유지보수: 효율적이고 깨끗한 코드를 작성하고 기존 시스템을 유지보수

• 제품 개발: 새로운 기능 개발 및 기존 기능 개선

• 문제 해결: 기술적 문제를 신속하게 식별하고 해결

• 팀 협업: 다른 팀원 및 부서와의 긴밀한 협업을 통해 프로젝트 진행

• 기술 연구: 최신 기술 동향을 파악하고, 이를 제품에 적용

필수 스킬

• 프로그래밍 언어: Python, Java, C++, JavaScript 등 주요 언어에 대한 깊은 이해와 숙련도

• 소프트웨어 개발: 소프트웨어 설계, 개발, 테스트 및 배포 경험

• 데이터베이스 관리: SQL, NoSQL 데이터베이스 사용 경험

• 문제 해결 능력: 복잡한 문제를 분석하고 효과적인 솔루션을 설계하는 능력

• 협업 도구: Git, JIRA 등 협업 도구 사용 경험

자격 요건

• 학력: 컴퓨터 공학, 소프트웨어 공학 또는 관련 분야의 학사 학위 이상

• 경력: 소프트웨어 개발 경력 3년 이상

• 포트폴리오: 개발한 프로젝트나 GitHub 레포지토리 등 실무 경험을 보여줄 수 있는 자료

• 커뮤니케이션 스킬: 팀 내외부와의 원활한 의사소통 능력
