개요

구글 드라이브(Google Drive)는 파일 저장 및 공유를 위한 클라우드 기반 도구입니다. 이 가이드는 구글 드라이브를 처음 사용하는 직원들을 위해 주요 기능과 사용 방법을 설명합니다.

구글 드라이브 시작하기

계정 생성 및 로그인

  1. 구글 계정 생성: 구글 계정이 없는 경우, 구글 계정 생성 페이지에서 계정을 생성합니다.

  1. 로그인: 구글 드라이브에 접속하여 구글 계정으로 로그인합니다.

프로필 설정

  1. 프로필 사진 추가: 구글 계정 페이지에서 “개인 정보” 섹션을 클릭하고, 프로필 사진을 업로드합니다.

  1. 프로필 정보 입력: 이름, 연락처 등 프로필 정보를 입력합니다.

주요 기능 사용 방법

파일 및 폴더 관리

  1. 파일 업로드

• 구글 드라이브 홈페이지에서 “새로 만들기” 버튼을 클릭하고, “파일 업로드”를 선택합니다.

• 업로드할 파일을 선택하여 구글 드라이브에 업로드합니다.

  1. 폴더 생성

• “새로 만들기” 버튼을 클릭하고, “폴더”를 선택합니다.

• 폴더 이름을 입력하고 “만들기”를 클릭하여 폴더를 생성합니다.

  1. 파일 이동

• 파일을 클릭하고 드래그하여 원하는 폴더로 이동합니다.

• 또는, 파일을 우클릭하고 “이동”을 선택한 후, 대상 폴더를 선택합니다.

파일 공유

  1. 공유 설정

• 공유할 파일 또는 폴더를 우클릭하고 “공유”를 선택합니다.

• 공유 설정 창에서 사람들의 이메일 주소를 입력하고 권한을 설정합니다(보기, 댓글, 편집).

• “보내기” 버튼을 클릭하여 파일 또는 폴더를 공유합니다.

  1. 공유 링크 생성

• 파일 또는 폴더를 우클릭하고 “공유”를 선택합니다.

• “링크로 공유 가능”을 활성화하고, “링크 복사” 버튼을 클릭합니다.

• 링크를 이메일 또는 메시지로 공유할 수 있습니다.

파일 협업

  1. 문서 공동 편집

• 공유된 문서를 열고, 동시에 여러 명이 문서를 편집할 수 있습니다.

• 편집 내용은 실시간으로 반영되며, 변경 사항은 자동으로 저장됩니다.

  1. 댓글 추가

• 문서의 특정 부분을 선택하고, 도구 모음의 댓글 아이콘을 클릭합니다.

• 댓글을 입력하고 “댓글 달기” 버튼을 클릭하여 의견을 추가합니다.

파일 검색 및 정리

  1. 파일 검색

• 구글 드라이브 상단의 검색창에 키워드를 입력하고 Enter 키를 누릅니다.

• 검색 결과에서 원하는 파일을 클릭하여 엽니다.

  1. 필터 사용

• 검색창 오른쪽의 필터 아이콘을 클릭하여 파일 유형, 소유자, 위치, 수정 날짜 등을 기준으로 검색 결과를 필터링합니다.

  1. 스타 표시

• 자주 사용하는 파일이나 폴더를 우클릭하고 “스타 추가”를 선택하여 쉽게 접근할 수 있도록 합니다.

• 사이드바의 “스타 표시됨” 섹션에서 스타 표시된 파일과 폴더를 확인할 수 있습니다.

백업 및 복원

  1. 백업 설정

• 중요한 파일을 정기적으로 구글 드라이브에 백업합니다.

• 컴퓨터의 파일을 자동으로 백업하려면 구글 드라이브 데스크톱 앱을 설치하고 백업 설정을 진행합니다.

  1. 파일 복원

• 삭제된 파일은 “휴지통”에서 확인할 수 있습니다.

• 휴지통에서 파일을 우클릭하고 “복원”을 선택하여 파일을 복원합니다.
